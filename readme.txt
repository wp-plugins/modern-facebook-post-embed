== Installation ==

= From WordPress Platform =
1. Download and activate from the Plugins page of your WordPress site.

= From Wordpress WEBSite =
1. Upload 'modern-facebook-post-embed' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find new button in WordPress post/page editor or you can use shortcode.

or

1. Upload `modern-facebook-post-embed.zip` with your board plugins installer

### Shortcode

* Just use this shortcode to embedding facebook post: [mfb_pe url="" bottom=""]
* url="here paste facebook post link" required attribute.
* bottom="here enter value for margin bottom, numbers only, default is 30" not required attribute.

### Example

	[fb_pe url="https://www.facebook.com/FacebookDevelopers/posts/10152128760693553"]
	[fb_pe url="https://www.facebook.com/FacebookDevelopers/posts/10152128760693553" mbottom="50"]
Find other Example in plugins support page.
